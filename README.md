# 👀 ReadBetween.AI

Because your PDF hides things.

## 🚀 Overview

ReadBetween.AI is a GenAI-powered PDF assistant that allows users to upload documents and ask questions. It uses Retrieval-Augmented Generation (RAG) to provide context-aware answers.

## ✨ Features

* 📄 Upload and analyze PDFs
* 💬 Chat-based interface
* 🔍 Semantic search using FAISS
* 🤖 AI-powered answers using Gemini
* 👀 “Found between lines” source detection

## 🧠 Tech Stack

* Streamlit
* LangChain
* FAISS
* Google Gemini API
* HuggingFace Embeddings

## ▶️ Run Locally

```bash
git clone https://github.com/your-username/readbetween-ai.git
cd readbetween-ai
pip install -r requirements.txt
streamlit run app.py
```
